	Setup

1. Download and extract repo contents to folder of choosing
2. Download and install Node.js and Express.js and add to PATH variables
3. Initialise npm in same folder using CLI
4. Run npm i node, npm i express, npm i papaparse, npm i fs, npm i csvtojson using CLI once navigated to chosen directory
5. Using CLI, type node app.js to start server
6. Open browser and enter localhost:3000 in url bar. 
7. See table contents on index.html page if app.js working
